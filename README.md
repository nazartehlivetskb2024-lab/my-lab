# ArchStudio — Сайт-портфоліо архітектурної студії

Сучасний односторінковий сайт (Landing Page) для архітектурної студії, виконаний у мінімалістичному стилі з використанням ефектів "матового скла" (Glassmorphism).

## 📁 Структура проекту
* `index.html` — головна сторінка сайту.
* `css/main.css` — файли стилів сайту.
* `IMG/` — папка з усіма зображеннями проекту.

## 🛠 Використані технології
* **HTML5** — семантична розмітка.
* **CSS3** — стилізація, Flexbox, Grid та ефекти розмиття фону.

## 🚀 Як відкрити проект
1. Склонуйте репозиторій або завантажте ZIP-архів.
2. Відкрийте файл `index.html` у будь-якому браузері.
